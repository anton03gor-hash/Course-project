## Запуск проекта

Требования:

PHP 8.3 и выше
Composer (менеджер зависимостей PHP)
Node.js и npm (для зависимостей во фронтенде)
MySQL 8.0 или выше


## Установка

1. Клонировать репозиторий:
git clone https://gitlab.irkat.ru/22082/course-structure.git
 cd course-structure

2. Установить зависимости PHP:
composer install

3. Установить зависимости Node.js:
 npm install

4. Настроить окружение:

Скопировать файл .env.example в .env:
cp .env.example.env

Сгенерировать ключ приложения:
php artisan key:generate

Настроить подключение к базе данных в файле .env:

DB_CONNECTION=mysql
DB_HOST=mysql-8.0
DB_PORT=3306
DB_DATABASE=popitka3Sklad
DB_USERNAME=root
DB_PASSWORD=

5. Создать и заполнить базу данных:
php artisan migrate php artisan db:seed

6. Собрать фронтенд-ресурсы:
npm run build


## Запуск

1. Запустить приложение:
php artisan serve

2. Собрать проект:
npm run build

3. Перейдите в браузере по адресу: http://127.0.0.1:8000


## Разработка
Для разработки можно использовать команду:
 npm run dev
Эта команда запустит сервер разработки с автоматической перезагрузкой.

## Тестовые данные
Для генерации тестовых данных можно использовать следующие команды:

php artisan db:seed --class=DatabaseSeeder
php artisan db:seed --class=RoleSeeder
               
## Устранение неполадок
Если возникнут проблемы:

1. Проверьте версии PHP и MySQL

2. Убедитесь, что все расширения PHP установлены

3. Проверьте права доступа к папкам storage и bootstrap/cache

4. Очистите кэш:

php artisan config:clear
php artisan cache:clear
php artisan view:clear
